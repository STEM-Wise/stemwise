DISCLAIMER!!

The courses offered are designed specifically for students with little to no prior knowledge of the respective topics. We understand that everyone starts their learning journey from different points, and our courses are tailored to provide a solid foundation for beginners.


Please note that grade requirements may vary for each course. We kindly request you to carefully review the grade specifications provided for each course before signing up. Our courses are structured to run once a week for a duration of 5 weeks. At the end of the 5th week, students will have the opportunity to showcase their achievements through a project. This project serves as a platform to demonstrate the knowledge and skillset they have acquired during the course duration.


We look forward to embarking on this learning journey with you and witnessing the growth and accomplishments that await. Should you have any further inquiries or require additional information, please do not hesitate to reach out to us.

Introduction to Website Development GRADES 6 - 13

Web development involves the creative process of building websites, which can range from intricate designs and components to basic text displayed on the internet or an intranet. The objective of this course is to provide learners with the fundamental knowledge of HTML and CSS, enabling them to create simple webpages for personal purposes. Additionally, students will acquire skills in JavaScript, enabling them to manipulate the Document Object Model (DOM) and develop websites that are interactive and dynamic.

Fundamentals of Python Programming GRADES 6 - 13

Python is a versatile and widely-used language known for its simplicity and readability, making it an excellent choice for beginners. Python is widely used in real-life applications, powering websites and web apps, data science and machine learning, automation, scripting, scientific computing, game development etc.

Throughout this course, students will delve into the basics of Python programming. They will learn about variables, data types, conditional statements, loops, and functions. By the end of the course, students will have a solid understanding of Python syntax, its core features, and the ability to write Python programs independently.

Fundamentals of C Programming GRADES 8 - 11

C is a general-purpose computer programming language. It was created in the 1970s by Dennis Ritchie, and remains very widely used and influential. The code written in C language is simple and execution efficient. As a result, the development of various languages has been influenced by C language. These languages are C++, C#, Python, Java, JavaScript etc.

This course provides an introduction to programming in the C programming language, widely used for diverse applications, from robotics to operating systems. Students will learn to create practical programs in C and gain insights into computer functioning, shaping programming language rules. Key topics covered include data types, loops, control structures, input/output, and Arduino programming with C.
